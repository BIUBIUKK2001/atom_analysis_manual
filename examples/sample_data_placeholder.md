# Sample Data Placeholder

This repository does not ship microscopy image data by default.

Recommended practice:

- keep raw data outside the Git repository
- point the notebooks to local DM3 / DM4 / MRC / TIFF files
- use the synthetic fallback paths in the notebooks when testing the workflow without real data

If you later add a shareable example dataset, keep it small, document calibration clearly, and avoid bundling sensitive or unpublished data.
